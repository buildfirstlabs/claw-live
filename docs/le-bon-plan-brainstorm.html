Le Bon Plan — Brainstorm projet

Objectif: lancer un magasin bon plan rentable dans le 06, avec une machine acquisition locale via Snap/TikTok.

Budget: 60k€ Surface cible: 150–200m² Loyer: 2k–2.5k€ (hors élec/eau/assurance) Panier moyen de travail: 35€

1) Positionnement

  • “Des produits utiles à prix accessibles” (pas du bazar sans logique).
  • Mix: 60% mobilier/puériculture/maison + 40% rotation rapide.
  • Promesse: arrivages réguliers + prix qui se voient tout de suite.

2) Le nerf de la guerre

  • Local visible + parking + accès livraison.
  • Achat intelligent (pas de stock mort).
  • Cadence contenu quotidienne pour pousser le trafic.

3) Idées activation lancement (30 jours)

J-15

Teasing travaux + “premiers arrivages”.

Semaine 1

Ouverture soft + vidéos “prix comparés”.

Semaine 2

Live Snap/TikTok “arrivage du jour”.

Semaine 3-4

Packs anti-inflation + offres week-end.

4) KPI à piloter chaque semaine

  • Clients/semaine
  • Panier moyen
  • CA hebdo
  • Marge brute %
  • Rotation stock (jours)

5) Roadmap simple

Phase 1: Magasin + Snap/TikTok

Phase 2: Stabilisation ventes + stock

Phase 3: Click & collect

Phase 4: Site simple + duplication

6) Questions à trancher maintenant

  • Zone exacte prioritaire (Antibes vs périphérie)
  • Plafond loyer réel accepté
  • Top 20 catégories produits de départ
  • Rôle de chacun sur exploitation / contenu / sourcing
